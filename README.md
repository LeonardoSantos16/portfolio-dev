# Portfolio Pessoal - Leonardo dos Santos Mendes Ferreira

Este projeto foi desenvolvido para centralizar minhas experiências, projetos e habilidades técnicas.

---

## Tecnologias Utilizadas
* **Core:** React 18, Vite (build tool), TypeScript.
* **Estilização:** Styled Components, Styled Media Query (Design responsivo).
* **Roteamento:** React Router Dom v6.
* **Ícones:** Phosphor Icons, React Icons.
* **Integrações e Utilidades:**
    * Axios (Consumo de APIs).
    * EmailJS (Envio de e-mails via formulário).
    * Date-fns (Manipulação de datas).
    * React Markdown (Renderização de descrições técnicas).
* **Qualidade de Código:** ESLint, TypeScript ESLint.

---

## Estrutura do Projeto

O portfólio está organizado nas seguintes seções:

* **Home:** Apresentação principal e resumo de impacto.
* **About Me:** Minha jornada, trajetória acadêmica e experiência profissional.
* **My Tech Stack:** Exibição organizada por categorias (Linguagens, Frontend, Backend, Ferramentas) com indicação de níveis de proficiência.
* **Projects:** Listagem completa de projetos desenvolvidos.
* **Project Details:** Visão detalhada de cada projeto, destacando desafios e tecnologias.
* **Contact:** Formulário de contato e links diretos para redes profissionais.
